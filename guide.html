紹介 アプリを開く

3 ステップで始まります

初めての方向けの、いちばん短い使い方ガイドです。

1 🖥️

アプリを開く

ブラウザで LayCAT を開くだけ。インストール不要。

Chrome / Edge 推奨
2 📁

プロジェクトを作る

「＋ プロジェクトを作成」からフォルダ or クラウドを選ぶ。

迷ったらフォルダで OK
3 🎬

動画をアップして描き込む

ショット → 工程を選んで「＋ 動画」で素材を追加。

H.264 の mp4 / mov 推奨
それだけ。あとは触りながら覚えられます。以下は必要になったら参照するリファレンスです。

SUBMIT & NOTIFY

提出とチェック依頼の流れ

📤

提出は「＋動画」だけで OK

新版アップ → 自動で「チェック待ち」に。担当者に通知。

👤

チェック担当を設定

タスク上部の「チェック」で担当者を選ぶ。ここが未設定だと通知されません。

🔔

通知はベルで確認

右上の 🔔 を開くとメンション・チェック依頼が並ぶ。

📦

まとめて出したい時はサブミット

ヘッダの「サブミット」で複数ショットを一括提出。

FAQ

よくあるつまずき

動画が再生できない

Animation / ProRes などブラウザ非対応コーデックの可能性。H.264 の mp4 / mov に変換してから再アップしてください。

音が出ない

レビュー画面の 🔊 / 🔇 トグルと音量スライダを確認。ミュートや音量 0 になっていませんか。それでも出ない場合は元動画に音声トラックが無い可能性。

チェック依頼の通知が来ない

タスクの「チェック」担当者に自分が設定されているか確認。それから 🔔 の「🗘 最新」を押すか、アプリをリロードしてみてください。

他人と共有したい（フォルダ運用）

Google Drive / Dropbox の同期フォルダにプロジェクトを置き、共有相手にも同じフォルダを「フォルダを選ぶ」で開いてもらう。

データはどこに？

フォルダ運用：選んだフォルダの中の laycat.project.json と media/。R2 運用：Cloudflare R2 バケット。

他人のアップに気づけない

リアルタイム push は無いので、作業前に 🔔 → 「🗘 最新」 を押すか、アプリをリロードしてください。
