できること

日々のレビュー ＆ 進行管理のすべてを、ここに。

アップロード・指示・確認・提出まで、映像制作でよくやる作業がひととおり揃っています。

Task Page

1 カットの作業を、1 画面に。

カットの工程を開くと、動画・コメント・アノテーション・ステータス・担当がすべて 1 画面に集まります。 アップするたびに版（v1・v2…）が残り、履歴も辿れます。

  • アップロードは「＋ 動画」ボタンひとつ
  • 版ごとにコメント／判定を切替表示
  • ステータス・作業／チェック担当をその場で設定
Annotation

動画のコマに、直接「絵」で指示。

動画のフレームに直接ブラシで描き込めます。指示はレイヤーで分けられ、あとから編集・削除も可能。

  • レイヤーで指示を分類（演出／作画／修正）
  • ブラシ・消しゴム・トレーシングペーパー・ゴースト
  • 筆圧検知（ペンタブ対応）
Face Guide

顔の向きは、立体的に指定。

「顔の向きガイド」で、上下・左右・傾きを 3D 的に示せる LayCAT 独自機能。 描いた指示は自動で動画に埋め込まれ、再生すると重なって表示されます。

  • ドラッグで yaw / pitch / roll を直感操作
  • コメント欄にも縮小画像で記録
Progress

進捗は、複数グラフを横並びで。

エピソード別の円グラフを横に並べて、作品全体の進み具合が一目でわかります。 工程／ステータスをワンクリックで切替、担当者でしぼり込みも可能。

  • 工程グラフ／ステータスグラフを即切替
  • リテイクが多いエピソードのつまずきも見える
Review Queue

自分が見る番のカットだけを、一覧に。

チェック担当者ごとにタブが分かれ、自分の担当分だけを表示できます。 残り件数はバッジで見えて、見落としを防止。

  • 担当者タブで自分のチェック分だけ表示
  • 一覧のままステータスを変更可能
Reel

カットをつなげて、通しで確認。

複数のカットを並べて連続再生。並び替えて通しのテンポを確認したり、REEL 上で書いたコメントは各カットのタスクページに反映されます。

  • 試写や繋ぎ確認にそのまま使える
  • REEL 内でもアノテーション・ステータス切替可能

LayCAT の強み

「他のツールには無い」機能を、標準で。

レビュー ＆ 進行管理ツールは多くありますが、LayCAT はブラウザだけで軽快に動きながら、 EXR ネイティブ対応・クラウド共有まで揃えているのが特徴です。

🎨 EXR

多層 EXR を、ブラウザで直接プレビュー。

CG レンダーの中間ファイル（.exr）を、Nuke や After Effects なしで直接ブラウザで開けます。 AOV レイヤーを自動判定し、Cryptomatte まで色分けして表示。

  • Beauty / Depth / Normal / Motion Vector / UV / Position を自動判定
  • Cryptomatte（オブジェクト／アセット分離）に対応
  • 露出（EV）／ガンマ／Depth の黒白点・反転をその場で調整
  • 「🎨 全レイヤー」モーダルで全レイヤーをタイル一覧表示
  • 主要レンダラー（Nuke／Arnold／Redshift／V-Ray）の出力に対応
☁ Cloud

クラウドストレージで、場所を選ばず共有。

プロジェクトデータを Cloudflare R2（クラウドストレージ）に保存できます。 会社・自宅・出先どこからでも、同じデータをブラウザ 1 つで開ける環境に。

  • ホーム画面のタイル一覧から、参加中プロジェクトに即接続
  • 招待メンバーだけがアクセス可能（受付係 Worker が判定）
  • ファイルサイズ制限＋アクセス履歴で安全運用
  • 従来のフォルダ共有と、プロジェクト単位で使い分け可能
📁 Folder

フォルダ運用で、サーバーもアカウントも不要。

プロジェクトデータをローカルフォルダや Google Drive / NAS の共有フォルダに丸ごと置くだけで、 それがそのままチームの「1 つの正」になります。サーバー構築・月額課金・運営者経由のデータ流出、すべて不要。 他のツールはクラウドや自社サーバーへの接続が前提ですが、 LayCAT ではネット接続なしのフォルダ運用も、クラウド運用と同格の正式な使い方として扱われます。

  • プロジェクトフォルダに laycat.project.json と media/ が置かれるだけの素直な構造
  • Google Drive / Dropbox / OneDrive / NAS など任意の同期フォルダで OK
  • オフラインでも作業可能、同期が復活すれば自動で反映
  • Chrome / Edge の File System Access API で、ブラウザから直接フォルダを読み書き
  • クラウド運用（R2）とプロジェクト単位で共存させられる

🎞 まとめ再生

通しで見ながら書いた指摘が、各カットページにそのまま残る。

複数カットをつなげて再生している最中に画面へ書き込んだ描き込みやコメントは、 それぞれのカットのレビュー履歴にも自動で残ります。 「試写のあとに 1 カットずつ開き直して書き写す」というよくある二度手間が発生しません。 逆に、各カットの画面からも「そのカットが含まれる連続再生」にワンクリックで戻れるので、 行き来がスムーズです。

  • 連続再生中の位置は、そのカットの正しいフレーム番号として記録されます
  • 試写 1 回ぶんの指摘を、全カットまとめて 1 クリックで送れます（担当者への通知もまとめて 1 通）
  • 各カット画面のサムネイルに「このカットが含まれる連続再生」への近道リンクが出ます
  • ローカルフォルダで運用しても、クラウドで運用しても、動きは同じです

他のツールでは「連続再生用のコメント」と「個別カットのコメント」が別々に保存されるのが一般的で、 あとから手動で書き写す必要があります。LayCAT ではどちらから書いても両方に残ります。

はじめ方

30 秒で使い始められます。

1

アプリを開く

右上の「アプリを開く」から、ブラウザで LayCAT を起動します。インストール不要。

2

プロジェクトを作成

フォルダ共有 or クラウド保存を選んで、プロジェクトを作成。カットや工程を追加します。

3

動画をアップ・レビュー

「＋ 動画」から素材をアップ。コマに描き込み、コメントし、OK / リテイクで判定するだけ。

4

チームで共有

フォルダを共有するか、クラウドプロジェクトにメンバーを招待すれば、すぐに共同運用できます。
