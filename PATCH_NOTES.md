# LayCAT 更新情報

## beta v0.0.3 （2026-07-21）

### 🎨 EXR フォーマット対応（新機能）
- **多層 EXR をブラウザで直接プレビュー可能に**（Nuke/After Effects なしでレンダー中間ファイルを確認できます）
- 自作 EXR パーサ搭載：ZIP / ZIPS / 無圧縮、HALF / FLOAT、Scanline / Tiled（Nuke・Arnold 書き出し）に対応
- **レイヤー自動判定して可視化**：
  - **RGB カラー**（beauty・diffuse 等）
  - **Depth**（Z）— グレースケール、黒点／白点／反転を手動指定可能
  - **Normal**（X/Y/Z）— カラーマップ
  - **Motion Vector**（角度→色相・強度→彩度）
  - **UV パス**（R=U, G=V）
  - **Position パス**（bbox 正規化）
  - **Cryptomatte**（プレビュー & ランク・ID ハッシュで色分け）
  - 輝度 (Y) / アルファ (A) の単色表示
- **アノテウィンドウにレイヤー切替＋露出／ガンマ／Depth 黒白点／反転** をコンパクトなスライダで搭載（数値直接入力も可能）
- 「🎨 全レイヤー」モーダル：EXR 内の全レイヤーをタイル一覧で俯瞰、クリックで直接開く
- 「🖼 サムネ更新」ボタン：任意レイヤー・設定でタスクページ一覧のサムネイルを更新
- 2 回目以降は **キャッシュ** で即座に開ける
- 非対応：Multi-part / Deep / PIZ・PXR24・DWA 圧縮

### 🌩 Cloudflare R2 保存対応（新機能）
- プロジェクトデータを **クラウドに保存できる** ように（従来のローカルフォルダ運用と併用可能）
- 新規プロジェクト作成モーダルに「ローカルフォルダ / Cloudflare R2」の選択肢を追加
- ホームの「⇄ プロジェクトを接続」に統合、参加可能な R2 プロジェクトを **タイル一覧** で表示
- **メンバー方式アクセス制御**：owner + members に登録された人だけがアクセス可能（`_access.json` 経由・Worker で 403 判定）
- Worker のセキュリティ強化：ファイルサイズ上限（JSON 10MB / メディア 500MB / 絶対 2GB）＋監査ログ
- セットアップ手順は `docs/R2_SETUP.md` 参照（Cloudflare 側の設定必要）

### 📝 タスクページ NOTE パネル（新機能）
- タスクページ右カラムに **リッチテキストのメモパネル** を追加
- 太字／斜体／下線／見出し（H₁・H₂）／文字サイズ／境界ライン
- 手動保存（**Ctrl+S** または「保存」ボタン）— 同期のズレを防ぐため
- 未保存のまま閉じようとすると警告
- 右側スライドではスペース確保のため自動的に非表示

### ✎ アノテーション操作の改善
- 描画／消しゴムを **1 つのトグルボタン**に統合（OFF → 描画 → 消しゴム → OFF と循環）
- ブラシサイズ／消しゴムサイズも **1 つのセレクト**に統合（現在のツールに応じて自動切替）
- **画像・EXR** で送信ボタンを押したとき、その瞬間の見た目（レイヤー切替中の状態＋描画）を **スクショとしてノートに焼き込み**（フレームの概念がない画像でもコメントにスクショが残るように）
- 画像アイテムのフレームコメント欄が正しく表示されるように

## beta v0.0.2 （2026-07-17）

### 再生範囲（IN/OUT）機能を追加
- **アノテーションウィンドウ**と **REEL** の両方に、動画の再生範囲を指定する機能を追加しました。
- タイムスライダで位置を動かして **`＜`** で開始（IN）、**`＞`** で終了（OUT）を設定、**`⌫`** で解除。
- 設定範囲はタイムラインに青の帯でハイライト。
- 再生ボタン押下時、プレイヘッドが範囲外なら自動で IN へジャンプ。
- 範囲設定中に OUT に到達したら、ループONなら IN へ戻ってループ、ループOFFなら OUT で自動停止。
- REEL では複数クリップを跨いだ範囲指定も可能。

## beta v0.0.1 （2026-07-17）

LayCAT の最初のパッチノート付きリリース。これまでに搭載された主な機能をまとめました。

### レビュー・アノテーション
- 動画・画像のフレーム単位のコメント／アノテーション
- ブラシ／消しゴム／顔の向きガイド／トレーシングペーパー／ゴースト表示
- ズーム／パン／回転／**左右反転**（`V` キーでトグル）
- **筆圧検知**（環境設定でON/OFF、既定OFF）
- **`Ctrl + 右クリック` パイメニュー**（描画・消しゴム切替）
- 比較再生（A/B・同期再生）
- 消しゴム操作を含む **Undo/Redo**

### タスク管理
- ショット・工程ごとのステータス管理（プロジェクトごとにカスタム定義可）
- 動画追加時に自動でステータスを「チェック待ち」に設定
- サブミット（複数カット一括提出、全体コメント＋カット個別コメント）
- メッセージ機能（誰でも削除可能、tombstone で復活防止）
- タスクページ／アノテウィンドウ／REEL 各所での動画・メッセージ削除

### 共有・コラボレーション
- フォルダ経由のプロジェクト共有（`laycat.project.json`）
- 3-way マージによる同時編集対応
- `laycat.project.json` 破損時の自動修復（末尾ゴミ除去）
- @メンション・通知

### REEL（シーケンスプレイヤー）
- 複数動画をクリップとして並べて連続再生
- REEL 内でもアノテーション・ステータス切替・左右反転可能

### アクセス制御
- Google ログイン（`access.json` の許可リストに基づく）
- プロジェクト個別パスワード保護（PBKDF2 → AES-GCM）
